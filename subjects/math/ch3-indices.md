# Chapter 3: Indices and Standard Form

## 3️⃣ Overview
Indices (exponents) are a shorthand for repeated multiplication. This chapter covers laws of indices, zero/negative indices, fractional indices, and standard form notation for very large and very small numbers.

---

## 📋 Topics Covered

1. Indices and exponents
2. Laws of indices
3. Zero and negative indices
4. Rational (fractional) indices
5. Standard form (scientific notation)

---

## 🔑 Key Formulas & Definitions

### Basic Definition
```
aⁿ = a × a × a × ... × a   (n times)

Where:
- a = base
- n = exponent/index
- aⁿ = power
```

### Example:
```
2⁵ = 2 × 2 × 2 × 2 × 2 = 32
```

---

## 📚 Five Laws of Indices

### **Law 1: Multiplication (Same Base)**
```
aᵐ × aⁿ = aᵐ⁺ⁿ
```
**Explanation**: When multiplying powers with the same base, ADD the exponents.

**Example**:
```
2³ × 2⁵ = 2³⁺⁵ = 2⁸ = 256
x² × x⁴ = x²⁺⁴ = x⁶
```

---

### **Law 2: Division (Same Base)**
```
aᵐ ÷ aⁿ = aᵐ⁻ⁿ
```
**Explanation**: When dividing powers with the same base, SUBTRACT the exponents.

**Example**:
```
2⁷ ÷ 2³ = 2⁷⁻³ = 2⁴ = 16
x⁶ ÷ x² = x⁶⁻² = x⁴
```

---

### **Law 3: Power of a Power**
```
(aᵐ)ⁿ = aᵐⁿ
```
**Explanation**: When raising a power to another power, MULTIPLY the exponents.

**Example**:
```
(2³)² = 2³ˣ² = 2⁶ = 64
(x²)⁵ = x²ˣ⁵ = x¹⁰
```

---

### **Law 4: Power of a Product**
```
aⁿ × bⁿ = (ab)ⁿ
```
**Explanation**: When both bases are raised to the same power, they can be combined.

**Example**:
```
2³ × 3³ = (2×3)³ = 6³ = 216
x² × y² = (xy)²
```

---

### **Law 5: Power of a Quotient**
```
aⁿ ÷ bⁿ = (a/b)ⁿ
```
**Explanation**: When dividing and both are raised to the same power, they can be combined.

**Example**:
```
6³ ÷ 2³ = (6/2)³ = 3³ = 27
x³ ÷ y³ = (x/y)³
```

---

## 🔢 Special Cases

### **Zero Index**
```
a⁰ = 1   (for any a ≠ 0)
```

**Explanation**: Any non-zero number raised to power 0 equals 1.

**Examples**:
```
5⁰ = 1
(-3)⁰ = 1
x⁰ = 1   (x ≠ 0)
```

**Why?** Using Law 2:  aⁿ ÷ aⁿ = aⁿ⁻ⁿ = a⁰ = 1

---

### **Negative Indices**
```
a⁻ⁿ = 1/aⁿ
```

**Explanation**: A negative exponent means "take the reciprocal".

**Examples**:
```
2⁻³ = 1/2³ = 1/8

5⁻² = 1/5² = 1/25

x⁻⁴ = 1/x⁴
```

**Combining negatives**:
```
(a/b)⁻ⁿ = (b/a)ⁿ
```

---

### **Fractional Indices**
```
a^(1/n) = ⁿ√a    (nth root)

a^(m/n) = ⁿ√(aᵐ)  = (ⁿ√a)ᵐ
```

**Explanation**: The denominator is the root; numerator is the power.

**Examples**:
```
4^(1/2) = √4 = 2

8^(1/3) = ³√8 = 2

27^(2/3) = ³√(27²) = ³√729 = 9
        = (³√27)² = 3² = 9
```

**Both forms are equivalent** — use whichever is easier!

## 📊 Summary Table: Laws of Indices

| Law | Formula | Example |
|-----|---------|---------|
| Multiplication | aᵐ × aⁿ = aᵐ⁺ⁿ | 2³ × 2² = 2⁵ |
| Division | aᵐ ÷ aⁿ = aᵐ⁻ⁿ | 3⁵ ÷ 3² = 3³ |
| Power of power | (aᵐ)ⁿ = aᵐⁿ | (2²)³ = 2⁶ |
| Product power | aⁿ × bⁿ = (ab)ⁿ | 2³ × 5³ = 10³ |
| Quotient power | aⁿ ÷ bⁿ = (a/b)ⁿ | 8² ÷ 4² = 2² |
| Zero | a⁰ = 1 | 99⁰ = 1 |
| Negative | a⁻ⁿ = 1/aⁿ | 2⁻³ = 1/8 |
| Fractional | a^(m/n) = ⁿ√(aᵐ) | 32^(2/5) = 4 |

## 🔬 Standard Form (Scientific Notation)

### Definition
```
A × 10ⁿ
```

Where:
- **A** is a number such that **1 ≤ A < 10**
- **n** is an integer (positive, negative, or zero)

### Rules
- **A must be between 1 and 10** (can equal 1, cannot equal 10)
- First significant figure must be before the decimal point
- All other digits come after the decimal

### Large Numbers (n is positive)
```
320,000 = 3.2 × 10⁵

5,400,000,000 = 5.4 × 10⁹
```

**How to find n**: Count how many places the decimal moves LEFT from the end.

### Small Numbers (n is negative)
```
0.0047 = 4.7 × 10⁻³

0.000032 = 3.2 × 10⁻⁵
```

**How to find n**: Count how many places the decimal moves RIGHT from the start.

---

## 📝 Conversion Examples

### Large Number → Standard Form
```
45,600,000

1. Place decimal after first digit: 4.5600000
2. Count decimal movement: 7 places left
3. Answer: 4.56 × 10⁷
```

### Small Number → Standard Form
```
0.00000342

1. Place decimal after first significant digit: 3.42
2. Count decimal movement: 6 places right
3. Answer: 3.42 × 10⁻⁶
```

### Standard Form → Ordinary Number
```
2.5 × 10⁴ = 25,000     (move decimal 4 places right)

7.3 × 10⁻³ = 0.0073    (move decimal 3 places left)
```

---

## 🧮 Operations with Standard Form

### Multiplication
```
(2 × 10³) × (3 × 10²) = (2 × 3) × 10³⁺² = 6 × 10⁵
```

### Division
```
(8 × 10⁵) ÷ (2 × 10²) = (8 ÷ 2) × 10⁵⁻² = 4 × 10³
```

### Addition/Subtraction
Convert to same power of 10 first:
```
2 × 10³ + 3 × 10² = 2000 + 300 = 2300 = 2.3 × 10³
```

## ✅ Exam Tips

1. **Memorize the 5 laws** - they're fundamental
2. **Watch the signs** - especially with negative exponents
3. **Check if answer is in standard form** - A must be 1 ≤ A < 10
4. **Show working clearly** - especially when using multiple laws
5. **Use calculator wisely** - know when to use it and when to work by hand
6. **Simplify fully** - don't leave answers with negative exponents if not required
