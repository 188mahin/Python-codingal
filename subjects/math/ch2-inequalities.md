# Chapter 2: Simultaneous Linear Inequalities

## 2️⃣ Overview
Simultaneous linear inequalities involve finding values that satisfy two or more inequality conditions at the same time. This chapter covers solving methods and representing solutions on number lines.

## 🔑 Key Symbols

```
>    Greater than
<    Less than
≥    Greater than or equal to
≤    Less than or equal to
```

---

## 📐 Number Line Representation

### Open Circle (○) vs Closed Circle (●)

```
x > 3   →  Open circle at 3, arrow pointing right  ○——→
x ≥ 3   →  Closed circle at 3, arrow pointing right  ●——→

x < 5   →  Open circle at 5, arrow pointing left  ←——○
x ≤ 5   →  Closed circle at 5, arrow pointing left  ←——●
```

**Explanation**:
- **Open circle (○)**: The value is NOT included
- **Closed circle (●)**: The value IS included

## 🧮 Solving Single Inequalities

### Rules (Same as Equations, with One Exception!)
```
When you multiply or divide both sides by a NEGATIVE number,
FLIP the inequality sign!
```

### Examples:

**Example 1**: x + 5 > 12
```
x + 5 > 12
x > 7          (no sign change - just subtract)
```

**Example 2**: -2x < 8
```
-2x < 8
x > -4         (FLIP sign when dividing by -2)
```

**Example 3**: 3x - 4 ≤ 11
```
3x - 4 ≤ 11
3x ≤ 15
x ≤ 5
```

## 🔗 Simultaneous Inequalities

### Strategy

**Step 1**: Solve each inequality separately  
**Step 2**: Draw both solutions on a number line  
**Step 3**: Find the OVERLAPPING region  
**Step 4**: Write the combined solution

---

### Example 1: No Overlap

```
Solve: x > 3  AND  x < 1

Inequality 1: x > 3   →  ○————→
Inequality 2: x < 1   →  ←————○

Number line:      1        3
                  ←——○  ○——→
                     ✗ No overlap!

Solution: NO SOLUTION
```

---

### Example 2: Partial Overlap

```
Solve: x > 1  AND  x ≤ 5

Inequality 1: x > 1   →  ○————————→
Inequality 2: x ≤ 5   →  ←————●

Number line:      1         5
                  ○————————●
                  1 < x ≤ 5

Solution: 1 < x ≤ 5  (or x ∈ (1, 5])
```

---

### Example 3: Multiple Overlaps

```
Solve: x > 2  AND  x ≥ -1

Inequality 1: x > 2    →  ○————————→
Inequality 2: x ≥ -1   →  ●————————→

Number line:    -1        2
                ●————————○————————→
                   x ≥ -1 AND x > 2

Solution: x > 2  (the stricter condition)
```

---

### Example 4: Full Overlap

```
Solve: x > 3  AND  x > 5

Number line:      3         5
                  ○————————○————→
                           (stricter)

Solution: x > 5
```

## 📊 Common Patterns

| Pattern | Solution |
|---------|----------|
| a < x < b | x is between a and b (excluding endpoints) |
| a ≤ x ≤ b | x is between a and b (including endpoints) |
| a < x ≤ b | x is between a and b (left open, right closed) |
| x < a AND x > b | NO SOLUTION (if a < b) |
| x > a OR x > b | x > min(a,b) |

## 🎯 Worked Examples

### Example 1: Simple Case
```
Solve: 2x + 1 > 5  AND  x - 2 ≤ 3

Step 1: Solve first inequality
2x + 1 > 5
2x > 4
x > 2

Step 2: Solve second inequality
x - 2 ≤ 3
x ≤ 5

Step 3: Find overlap
x > 2 AND x ≤ 5

Solution: 2 < x ≤ 5
```

---

### Example 2: With Negative Numbers
```
Solve: -3x < 12  AND  x + 5 > 2

Step 1: Solve first inequality
-3x < 12
x > -4  (FLIP sign when dividing by -3)

Step 2: Solve second inequality
x + 5 > 2
x > -3

Step 3: Find overlap
x > -4 AND x > -3

Solution: x > -3 (the stricter condition)
```

---

### Example 3: Three Inequalities
```
Solve: x > 1  AND  x ≤ 7  AND  x ≠ 4

Step 1: First two inequalities
1 < x ≤ 7

Step 2: Exclude x = 4
Solution: 1 < x ≤ 7, x ≠ 4
          or (1, 4) ∪ (4, 7]
```

## 📈 Integer Solutions

When asked for "integer solutions", list all whole numbers in the valid range.

```
Example: Find integer solutions for 2 < x ≤ 5

Solution: x ∈ {3, 4, 5}
(Not 2, because 2 is not > 2)
```

---

## ✅ Exam Tips

1. **Always solve each inequality separately first**
2. **Draw number lines carefully** - use ○ and ●
3. **Identify the OVERLAP region** - don't just combine
4. **Remember to FLIP the sign** when multiplying/dividing by negatives
5. **Show working clearly** - write both solved inequalities
6. **Check your answer** - substitute a test value
7. **Write final answer clearly** - use notation like: a < x ≤ b

---

## 🧪 Verification Method

Always check your answer by testing values:

```
For solution 2 < x ≤ 5:

Test x = 1.5: Does NOT satisfy (too small) ✗
Test x = 3:   Satisfies both inequalities ✓
Test x = 5:   Satisfies (= is included) ✓
Test x = 6:   Does NOT satisfy (too large) ✗
```
