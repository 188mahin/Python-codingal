# Chapter 1: Quadratic Equations and Functions

## 1️⃣ Overview
Quadratic equations are polynomial equations of degree 2. This chapter covers solving methods, graphing, and real-world applications.

---

## 📋 Topics Covered

1. Solving quadratic equations by completing the square
2. Solving quadratic equations by using the quadratic formula
3. Solving quadratic equations by graphical method
4. Solving fractional equations reducible to quadratic equations
5. Sketching graphs of quadratic functions
6. Applications of quadratic equations in real-world contexts

---

## 🔑 Key Formulas

### Standard Form
```
ax² + bx + c = 0   (where a ≠ 0)
```

### Completing the Square
```
x² + bx = (x + b/2)² - (b/2)²

ax² + bx + c = a(x + b/2a)² + (c - b²/4a)
```

### Quadratic Formula
```
x = (-b ± √(b² - 4ac)) / 2a
```

**Discriminant (Δ)**: Δ = b² - 4ac
- If Δ > 0: Two distinct real roots
- If Δ = 0: One repeated real root (equal roots)
- If Δ < 0: No real roots (complex roots)

## 🎯 Solving Methods

### Method 1: Factorisation
Use when the quadratic factors nicely.

**Example**: x² + 5x + 6 = 0
```
(x + 2)(x + 3) = 0
x = -2 or x = -3
```

✅ **Fastest method if it works**
❌ **Doesn't work for all equations**

---

### Method 2: Completing the Square
Always works and gives the vertex form.

**Example**: x² + 6x + 5 = 0
```
x² + 6x + 5 = 0
x² + 6x = -5
x² + 6x + 9 = -5 + 9
(x + 3)² = 4
x + 3 = ±2
x = -3 ± 2
x = -1 or x = -5
```

✅ **Works for all quadratics**
✅ **Gives vertex form directly**

---

### Method 3: Quadratic Formula
Universal method for all quadratic equations.

**Example**: 2x² - 7x + 3 = 0
```
a = 2, b = -7, c = 3

x = (7 ± √(49 - 24)) / 4
x = (7 ± √25) / 4
x = (7 ± 5) / 4

x = 3 or x = 0.5
```

✅ **Always works**
✅ **Fastest for most problems**

---

### Method 4: Graphical Method
Plot the graph and read x-intercepts.

**Example**: x² - 4x + 3 = 0
```
Plot y = x² - 4x + 3
The curve crosses the x-axis at x = 1 and x = 3
Therefore: x = 1 or x = 3
```

✅ **Visual understanding**
❌ **Less precise than algebraic methods**

## 📊 Sketching Graphs of Quadratic Functions

### Vertex Form
```
y = a(x - h)² + k
```
- **Vertex**: (h, k)
- **Line of symmetry**: x = h
- If a > 0: parabola opens upward (minimum point)
- If a < 0: parabola opens downward (maximum point)

### Factored Form
```
y = a(x - p)(x - q)
```
- **x-intercepts**: x = p and x = q
- **Line of symmetry**: x = (p + q)/2
- **Vertex**: ((p + q)/2, y-value at that x)

### Standard Form
```
y = ax² + bx + c
```
- **y-intercept**: (0, c)
- **Line of symmetry**: x = -b/2a
- **Vertex**: (-b/2a, y-value at that x)

## 🔄 Fractional Equations Reducible to Quadratic

### Steps:
1. Identify the LCD (Least Common Denominator)
2. Multiply both sides by the LCD
3. Expand and simplify to get a quadratic
4. Solve the quadratic equation
5. **Check**: Ensure answers don't make any denominator zero

### Example:
```
1/(x-1) + 2/(x+1) = 3

LCD = (x-1)(x+1)

(x+1) + 2(x-1) = 3(x-1)(x+1)
x + 1 + 2x - 2 = 3(x² - 1)
3x - 1 = 3x² - 3
0 = 3x² - 3x - 2
3x² - 3x - 2 = 0

Using quadratic formula:
x = (3 ± √(9 + 24)) / 6 = (3 ± √33) / 6
```

⚠️ **Important**: Always check that your solutions don't make the original denominators zero!

## 🌍 Real-World Applications

### Common Contexts:
- **Projectile motion**: Height = -gt²/2 + v₀t + h₀
- **Area problems**: Rectangle with dimensions involving x
- **Revenue problems**: Profit = Revenue - Cost (both often quadratic)
- **Physics**: Distance, velocity, acceleration relationships

### Example Problem:
```
A ball is thrown upward from ground level with initial velocity 20 m/s.
Its height h(t) = -5t² + 20t

When does it hit the ground?
-5t² + 20t = 0
-5t(t - 4) = 0
t = 0 (initial) or t = 4 (lands)

Answer: 4 seconds
```

## ✅ Exam Tips

1. **Read carefully** - identify which method is best
2. **Show working** - write all steps clearly
3. **Check discriminant** first - know how many solutions to expect
4. **Verify answers** - substitute back into original equation
5. **For fractional equations** - don't forget to exclude invalid solutions
6. **State the equation** - write ax² + bx + c = 0 before solving
